## Настройка среды разработки
___
В виртуальном окружении необходимы следующие пакеты:
- pip 22.0.2
- distro 1.7.0 
- wheel 0.37.1 
- cryptography 3.4.8 
- Pillow 9.0.1 
- keyring 23.5.0 
- PyYAML 5.4.1. 
- setuptools 59.6.0 
- pyparsing 2.4.7 
- zipp 1.0.0 
- future 0.18.2

Для установки зависимостей в новое окружение используйте `pip install -r requirements.txt`
